# ai-101
人工智能通识

## 课程简介

本课程旨在帮助学习者建立对人工智能的系统性认知，从基础概念到前沿技术，循序渐进地掌握 AI 核心知识体系。

## 课程大纲

### 第一部分：人工智能基础
- 什么是智能
- AI 发展简史
- 数据与算力基础
- 模型的本质

### 第二部分：机器学习
- 监督学习与非监督学习
- 特征工程
- 模型评估与优化
- 经典算法实践

### 第三部分：深度学习
- 神经网络基础
- 卷积神经网络 (CNN)
- 循环神经网络 (RNN)
- 模型训练技巧

### 第四部分：自然语言处理 (NLP)
- 文本表示与词向量
- 序列模型
- 注意力机制
- NLP 核心任务

### 第五部分：大语言模型 (LLM)
- Transformer 架构
- 预训练与微调
- Prompt Engineering
- LLM 应用实践
