# 版式与样式参考

## 1. 基础画布
- 默认采用 16:9 比例（推荐 1600x900 或 1920x1080 思路设计）。
- 页面四周保留安全边距，避免投屏裁切。
- 左上放标题，主体内容纵向居中或上对齐，不要贴底。

## 2. 投屏可读字号
- 封面主标题：48-88px（Tailwind 建议 `text-6xl` 到 `text-8xl`）。
- 内容页标题：44-64px（`text-5xl` 到 `text-7xl`）。
- 正文与要点：24-36px（`text-2xl` 到 `text-4xl`）。
- 备注/脚注：18-20px（`text-lg` 到 `text-xl`）。

## 3. 信息密度
- 每页仅 1 个核心结论。
- 每页 3-5 条要点，单条尽量 10-14 个词。
- 一页内同时出现文本+图时，文本区域尽量不超过页面宽度的 55%。

## 4. 配色风格（参考示例图）
视觉方向：深色背景 + 高亮绿色强调 + 冷灰文字层级。

推荐主色：
- 背景 `#0b0d10`
- 面板 `#111318`
- 主文字 `#f3f4f6`
- 次文字 `#a1a1aa`
- 强调绿 `#3ecf8e`
- 强调绿悬停 `#2fb67b`
- 边框灰 `#22262e`

## 5. Tailwind 主题用法
在单 HTML 中，优先使用 Tailwind Play CDN，并内联主题：

```html
<script src="https://cdn.tailwindcss.com"></script>
<script>
  tailwind.config = {
    theme: {
      extend: {
        colors: {
          deck: {
            bg: '#0b0d10',
            panel: '#111318',
            text: '#f3f4f6',
            muted: '#a1a1aa',
            accent: '#3ecf8e',
            accentHover: '#2fb67b',
            border: '#22262e'
          }
        }
      }
    }
  }
</script>
```

## 6. 图形与图片策略
- 优先使用内联 SVG（图标、示意图、柱状图、流程图）。
- 避免外部网络图片依赖，确保单文件可分发。
- 图形最小可视尺寸建议不低于 240x140。
- 文字叠加到图形上时，保持足够对比度。

## 7. 动效规则
- 仅使用翻页级动效（切换/淡入）。
- 动效时长建议 180-260ms。
- 不对每个元素单独做复杂动画，避免分散注意力。

## 8. 最终检查清单
- 不出现空标题、空要点或断行异常。
- `1920x1080` 投屏可读，`1366x768` 仍不重叠。
- 首末页结构完整（开场目标 + 结束行动）。
- 箭头键翻页、Home/End 跳转可用。
